    The cvxopt package is needed for the python codes. You could find the detailed 
information in the following links:

    http://cvxopt.org/index.html
    (CVXOPT: Python Software for Convex Optimization)

    You could download the package from the above link. Be sure to install the numpy
and MKL packages correctly. For Windows users, there is another simpler way to install 
the pacakge by using the following Python Extension Packages:

    http://www.lfd.uci.edu/~gohlke/pythonlibs/#cvxopt

    The user's guide with samples could be found in the "CVXOPT" website, and there is 
another fundamental instruction in the following link:

    https://courses.csail.mit.edu/6.867/wiki/images/a/a7/Qp-cvxopt.pdf
